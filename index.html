소개

소프트웨어과 전문동아리 중 유일하게 웹동아리인 IWOP은 19년의 역사를 자랑하는 동아리 입니다. 우수한 수상실적을 보유하고있으며, 웹 클라이언트, 서버 웹 게임 개발과 웹 디자인을 배울수 있습니다!

커리큘럼

code

1.FrontEnd Basic

처음 IWOP에 들어오면 HTML5와 CSS3로 정적 페이지를 만들어 봅니다.

description

2.Javascript

자바스크립트로 동적인 홈페이지를 제작해봅니다.

http

3.Node.js

Node.js와 express프레임워크로 웹서버를 만들어 봅니다.

storage

4.Database, server

MongoDB,Socket.io,Passport로 다양한 서버를 제작해봅니다.

view_compact

5. FrontEnd Advanced

Vue 로 SPA를 만들어 실제로 배포해봅니다.

videogame_asset

6. Web Game

phaser.js로 웹게임을 개발하고 기획해봅니다.

QnA

Q 아이왑에 지원하려면 실력이 얼마나 좋아야 하나요?
A 완전 처음 이여도 상관없습니다! 웹을 배우고자 하는 열정만 있다면 충분합니다!
Q 포트폴리오를 반드시 내야 하나요?
A 내든 안내든 상관없으나 내면 좋습니다.
Q 지원 자격이 어떻게 되나요?
A 선린 1학년 소프트웨어과,멀티미디어과가 지원 가능합니다!
Q 5.기(점오기)뽑나요?
A 아쉽게도 지원을 받지 않습니다.
